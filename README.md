# sprint_eng_software-code-ops
Uma solução com inteligência artificial desenvolvida especificamente para o ecossistema do Metaindústria (Challenge 2026 FIAP × SPI Integração). O objetivo central é transformar a gestão de Saúde e Segurança do Trabalho (SST) no chão de fábrica, realizando a transição definitiva do modelo tradicional  para um modelo proativo e preventivo.
